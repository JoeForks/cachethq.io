What is {{ site.project_title }}?

{{ site.about }} {{ site.project_title }} is a Simple to use, easy to manage, Status Page project. Ever had problems making sure users know whats going on with your systems. Ever wish you would stop getting email asking if your service is down. No Problem! {{ site.project_title }} can be easily installed and used to track third party service, and let your users know whats going on!

What does it Do?

  • Easily lets you manage Components
  • Track Incidents on a Per day basis
  • Provides a Powerful management portal
  • Integrates with Third party services, for Inbound/Outbound Incident reporting
  • People can subscribe to receive notifications of Incident updated

How does it work?

{{ site.project_title }} is a PHP Powered application (Built using Laravel 4.3) which utilises Twitter bootstrap for the User Interface. So it is built upon well used, and tested Open Source Projects. {{ site.project_title }} can be used to input Incidents which have occured, which will be displayed on the status page, which gives a clear and transparent overview of your systems status. It will also monitor for any changes to the status, and will automatically push out to Third Party Service, so your customers (and your staff) can know as an Incident occurs or changes status.
